## 介绍

### 学习目标

- 了解强化学习问题及其与监督学习的区别

### 摘要
- 强化学习（RL）是一种基于目标的学习方式。
- 在RL中，智能体通过与环境交互获得的经验来学习。 在监督学习中，我们不能影响环境。
- 在RL奖励中，奖励通常会在时间上延迟，并且智能体会尝试使得长期收益最大化。 例如，人们可能需要做出看似次优的动作以达到游戏中的获胜位置。
- 智能体通过状态，动作和奖励与环境交互。
